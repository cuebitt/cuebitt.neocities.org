Open Source Attribution

The Web Component Shop page is built using the following open-source libraries:

  • PicoCSS
  • Bootstrap Icons
  • ComcastSamples/wc-toast

The dependencies above are used to create this web page, but not the components themselves. The web components provided do not have any dependencies, though future ones might. Any future components with dependencies will be clearly labeled as such.

Crediting Web Component Shop

Source code for all the web components provided is hosted on GitHub at cuebitt/webcomponents.
