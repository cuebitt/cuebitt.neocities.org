Home Page

This website's home page is designed to look like a Nintendo Wii console's home menu. It uses sound effects that are sourced from a Wii console, which were made by Nintendo. Some of the CSS styles are borrowed from Wii Menu by Alexis Oulès on Figma, which is distributed under the CC-BY-4.0 License.

The home page also uses the following open source libraries:

  • @floatingui/dom (MIT), used for tooltips
  • Howler (MIT), used to play sound effects
  • @splidejs/splide (MIT), used for the channel carousel
  • modern-normalize (MIT)
  • nanoid(MIT)
